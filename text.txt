## Konteks untuk Chat Baru

---

**OTTO-AI — Mulai dari nol**

Struktur folder sudah dibuat di `/data/asd/otto-ai/`

**Filosofi Otto:**
- Otto tidak punya file yang mendikte siapa Rofi
- Otto mengamati perilaku → bertanya untuk mengenal lebih dalam → membentuk hipotesis → bertanya untuk verifikasi
- Jika jawaban salah → revisi model internal
- Otto belajar seperti manusia mengenal manusia baru

**Yang sudah ada:**
- `.env` dengan API keys (Groq + GitHub(tapi belum diisi untuk folder baru ini))
- `voices/id_ID-news_tts-medium.onnx` model TTS
- Semua folder kosong siap diisi

**Stack teknis:**
- OS: openSUSE Tumbleweed, Hyprland, Wayland
- Audio: PipeWire (pw-record, pw-play), speaker laptop sink ID 58
- STT: faster-whisper (tiny untuk perintah pendek, medium untuk panjang)
- LLM: Groq API (llama-3.1-8b-instant untuk perintah, llama-3.3-70b untuk ngobrol)
- TTS: Piper binary di `/usr/local/bin/piper`
- Server: FastAPI + uvicorn port 8000, HTTPS self-signed SSL
- Interface: WebSocket dari iPhone chrome

**Mulai dari: `core/config.py` dulu**
